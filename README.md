# Corotector
* #### 장르 : VR 슈팅 게임
* #### 역할 : 프로그래머
* #### 개발 언어, 툴 : C#, Unity
* #### 사용 기기 : HTC Vive

## 프로젝트 소개
### 게임 소개
* 제 16회 앱잼에서 제작되었습니다.
* 해당 게임은 VR 슈팅 게임입니다.
* 오른쪽 탁자에 놓여있는 폭죽은 사용시 리필되며 폭죽은 왼쪽 모닥불을 이용해 사용이 가능합니다.
* 정면에서는 적들이 몰려오며, 이를 폭죽으로 막아야합니다.
* 막지못한채 플레이어에게 접근하게 될 시 게임 오버됩니다.
+ **폭죽**
  + 연발형 폭죽 : 데미지가 낮은 폭죽 여러개가 매우 빠르게 발사됩니다.
  + 막대기 폭죽 : 데미지는 높지만 공격속도가 느린 폭죽으로, 총 3발 정도가 발사됩니다.
